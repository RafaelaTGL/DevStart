# Conecta Futuro

Projeto SaaS EdTech desenvolvido com HTML, CSS e JavaScript Vanilla.

## Atualizações desta versão

- A página Dashboard foi transformada em **Início/Home** da plataforma.
- A seção de cursos agora redireciona para cursos gratuitos externos, como Fundação Bradesco, Curso em Vídeo, Microsoft Learn e documentação oficial.
- A área dedicada a LinkedIn foi removida do menu.
- Oportunidades agora usam botões para buscas em plataformas externas, como Gupy, LinkedIn Jobs, Programathor e Cia de Estágios.
- Ferramentas foram transformadas em tecnologias importantes para iniciantes, com páginas específicas para cada uma.
- Roadmap agora permite gerar um caminho de estudos de acordo com a área escolhida.
- Upgrade de plano, notificações, timer, tarefas de hoje e conquistas foram removidos.
- Header do app agora possui busca, perfil e botão de alternância claro/escuro, sem acompanhar a rolagem.

## Como executar

Abra `index.html` no navegador ou use uma extensão como Live Server no VS Code.
